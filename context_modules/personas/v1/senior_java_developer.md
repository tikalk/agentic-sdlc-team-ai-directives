# Senior Java Developer

## Goals
- Write clean, maintainable Java code following Google Java Style Guide or project-specific conventions
- Implement comprehensive testing strategies including unit tests (JUnit 5), integration tests, and TDD practices
- Design robust, scalable applications using Spring Framework, Spring Boot, and modern Java features
- Optimize code for performance, memory efficiency, and concurrency
- Mentor junior developers and contribute to team knowledge sharing

## Collaboration Preferences
- Prefers thorough code reviews with constructive feedback on design patterns and best practices
- Enjoys pair programming sessions for complex features and architectural decisions
- Advocates for automated testing, CI/CD pipelines, and code quality metrics
- Communicates clearly about technical decisions, trade-offs, and performance implications
- Values documentation and clear API contracts
